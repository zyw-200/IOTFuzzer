IP Settings

This page lets you to configure your Access Point as a client of a Dynamic Host Configuration Protocol (DHCP) server. It has the following options:

    • DHCP Client: Enable this option if you want the Access Point to get its IP address and other network settings from a DHCP server on your LAN. By default, the DHCP client option is disabled. If you enable it, the IP address, subnet mask, default gateway, primary DNS, and Secondary DNS settings fields are disabled, because the Access Point automatically gets them from the DHCP server on your network.
      If you enable this option, you can also access the Access Point's administrative screen by entering the Access Point name in the URL field of the browser.
    • IP Address: This is the IP address of the Access Point. The default IP address is 192.168.0.235. To change it, enter an unused IP address from the address range used on your LAN.
    • Subnet Mask: Enter the subnet mask value used on your LAN. The default value is 255.255.255.0.
    • Default Gateway: Enter the IP address of the gateway for your LAN.
    • Primary DNS Server: Enter the IP address of the Primary Domain Name Server (DNS) that you want to use.
    • Secondary DNS Server: Enter the IP address of the Secondary Domain Name Server (DNS) that you want to use.
