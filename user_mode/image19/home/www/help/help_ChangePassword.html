Change Password Help

This page enables you to change the Access Point administrator's password.

To change the password:

    • Type the old password.
      The default password for the Access Point is: password.
    • Type a new password and type it again in the Repeat new password box to confirm it.
      Note: Be sure to write it down in a secure location.
    • Select Yes for Restore Default Password.
    • Click Apply to have the password changed or click Cancel to keep the current password.
