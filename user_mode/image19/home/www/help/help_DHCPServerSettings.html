DHCP Server Settings

Your Access Point comes with an in-built DHCP Server for Wireless Clients only, which can be especially useful in small networks. This page lets you to enable and configure the DHCP server. It has the following options:

    • DHCP Server: Enable this option for the Access Point to function as a DHCP Server for Wireless Clients only. The Access Point will provide the pre-configured TCP/IP configurations for all wireless stations connected to this Access Point.
    • DHCP Server VLAN ID: Enter DHCP server VLAN ID. The VLAN ID range is between 1 and 4094. DHCP server will service this VLAN.
    • Starting IP Address: Enter the starting IP address of the range that can be assigned by the DHCP Server.
    • Ending IP Address: Enter the ending IP address of the range that can be assigned by the DHCP Server.
    • Subnet Mask: Enter the subnet mask that will be assigned to the Wireless Clients by the DHCP Server.
    • Gateway IP Address: Enter the IP address of the default network gateway for all traffic beyond the local network.
    • Primary DNS Server: Enter the IP address of the primary DNS server for the network.
    • Secondary DNS Server: Enter the IP address of the secondary DNS Server for the network.
    • Primary WINS Server: Enter the IP address of the primary WINS Server for the network.
    • Secondary WINS Server. Enter the IP address of the secondary WINS Server for the network.
    • Lease: The lease time for the assigned IP for the wireless client. The client must renew the lease based on the lease period.
