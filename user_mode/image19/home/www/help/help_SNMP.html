SNMP

Enable SNMP to allow the SNMP network management software, such as HP OpenView, to manage the wireless access point by using SNMPv1/v2 protocol.

    • Public Community Name : The community string to allow the SNMP Manager to read the wireless access point's MIB objects.
    • Private Community Name : The community string to allow the SNMP Manager to read and write the wireless access point's MIB objects.
    • Community Name for Traps : The community name which is associated with the IP address to receive traps.
    • IP address to Receive Traps : The IP address of the SNMP Manager to receive traps sent from the wireless access point.
    • Trap Port: The port number on which the SNMP Manager will receive traps.
