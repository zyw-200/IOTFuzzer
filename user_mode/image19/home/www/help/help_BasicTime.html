Basic Time Settings Help

Time Settings

This page lets you configure the time-related settings of your Access Point.

Configure the following information:

    • Time Zone: Select the appropriate local time zone for your region or country.
    • NTP Client: Enable this to use a Network Time Protocol (NTP) server to synchronize the clock of your Access Point. Disable this option if you do not want to use an NTP server.
    • Use Custom NTP Server: Check this box if you wish to use an alternate NTP Server. By default, the NETGEAR NTP server is used by the Access Point.
    • Hostname / IP Address: Provide the host name or IP address of the NTP server, if you are using a custom NTP server.
