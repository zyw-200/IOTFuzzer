Basic Wireless Settings Help

Wireless Settings

This page lets you configure the basic Wireless settings for your Access Point.

Specify the following information to configure the Wireless settings of your Access Point:

    • Wireless Mode: Specify the desired wireless mode. In addition to the default 802.11ag mode, you can also choose 802.11a or 802.11an modes. In the 802.11an mode, both 802.11n- and 802.11a-compliant devices can be used with this Access Point.
      If you select this option and if other settings on this screen are disabled, then you must select the Turn Radio On radio button to enable available options on this screen.
    • Turn Radio On: Disable this option to disable wireless access for the selected mode. To disable all wireless access through this Access Point, you must turn off the 802.11b/g/n, as well as the 802.11a/n radios. At any given point, you can only have 802.11a/n radio on.
    • Wireless Network Name (SSID): Enter a unique, case-sensitive, and 32-character-long alphanumeric wireless network name. Any wireless device that needs to connect to this Access Point must specify this SSID to do so. The default SSID is NETGEAR_11a.
    • Scheduler Status: This information is to display the status of Scheduled Wireless ON-OFF feature. When this feature is enabled, here it will show 'ON' in green colour and user will not be able to change radio status. When this feature is disabled, here it will show 'OFF' in red colour.
    • Broadcast Wireless Network Name (SSID): Specify whether you want to broadcast the SSID of this Access Point. If you disable this option, then only devices that know the SSID can connect to this Access Point. However, this also means that wireless network “discovery” feature of some products will not work with this Access Point.
      By default, broadcasting the SSID is enabled. In this case, the Wireless Access Point will broadcast its SSID, allowing Wireless Stations which have a "null" (blank) SSID to adopt the correct SSID.
    • Channel/Frequency: Select the channel you wish to use on your wireless LAN. This field sets the operating frequency to use.
      You should not need to change the channel unless you notice interference problems, or if you are setting up this Access Point near another Access Point. NETGEAR recommends leaving this field to the default of Auto.
    • MCS Index/Data Rate: Select the available transmit data rates of the wireless network.
    • Channel Width: Select the available channel width of the Access Point. A wider channel improves the performance, but some legacy devices can only operate on either 20 MHz or 40 MHz.
    • Guard Interval: Select the value that will protect transmissions from interference. A shorter guard interval improves performance, but some legacy devices can only operate with a long guard interval.
    • Output Power: Select the available transmit power of the access point. This option sets the transmit signal strength of the Access Point. Increasing the power improves performance, but if two or more Access Points are operating in the same area, on the same channel, it can cause interference.
