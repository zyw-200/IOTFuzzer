General

This page lets you configure the cloud settings of your Access Point.

Cloud Enabled

This field is the configuration parameter which gives the cloud Agent running status. The options available are yes and no. If the cloud agent is currently running, then yes is selected otherwise no is selected. On the other hand if the user wants to stop the agent from running he needs to click on "no" and click on apply. If the user wants to run the cloud agent then the user needs to click on "yes" and apply. When the AP comes with factory defaults the agent was already running and the user can see "yes" already selected in the GUI.

When the agent is running, it is connected to the cloud online service, registers itself with the service and then onwards helps the online service (Cloud WMS) to configure the AP, monitor the AP and receive notifications from the AP.

Cloud Activation Status

There are two possible states defined for this value. This is a read only value in the GUI and displays the status of activation.

a) Pending - The activation cycle must be started or is underway. The cloud agent could be running or not running in this state. Usually when the AP comes up with factory defaults, this is the usual default state. The change of state is usually from pending to activated.

b) Activated - The cloud agent has successfully completed an activation cycle against the cloud online service. An AP when added to the cloud using Serial Number to a location and when the agent communicates with the cloud and registers itself, then the AP gets activated.

Cloud Connectivity Status

There are two possible states and it represents the connection state of the agent with the cloud online service

a) Connected - Agent is connected to the Cloud WMS and is currently communicating actively. The AP should be able to access internet for this.

b) Disconnected - Agent is not connected to the Cloud WMS is not able to communicate. The disconnection could happen due to cable plug off, internet down, unreachable gateway, DNS resolution problems etc.

The Link to Cloud UI or Cloud WMS is wireless.netgear.com

The user can use this link to create and access cloud WMS account and add the AP to the cloud if not already added. The AP can be managed using a cloud account and monitor it.

Access Point Information

The Access Point Information section displays information about the Access Point.

    • Access Point Name: The Access Point's NetBIOS name.
    • Ethernet MAC Address: The MAC Address of the Wireless Access Point's Ethernet port.
    • Wireless MAC Address for 2.4GHz: The MAC Address of the Wireless Access Point's wireless card when operating at 2.4GHz.
    • Wireless MAC Address for 5GHz: The MAC Address of the Wireless Access Point's wireless card when operating at 5GHz.
    • Country/Region: The domain or region for which the Wireless access point is licensed for use.
    • Firmware Version: The version of the firmware currently installed.
    • Serial Number: The Serial Number of the device.
    • Current Time: Displays the current system time of the access point.
