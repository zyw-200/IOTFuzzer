Backup Settings

Once you have the Access Point working properly, you should back up the information to have it available if something goes wrong. When you backup the settings, they are saved as a file on your computer. This section allows you to back up the Access Point’s current settings.

Click Backup to create a backup file of the current settings:

    • If you don't have your browser set up to save downloaded files automatically, then locate where you want to save the file, rename it if you like, and click Backup.
    • If you have your browser set up to save downloaded files automatically, the file is saved to the your browser's download location on the hard disk when you click Backup.
