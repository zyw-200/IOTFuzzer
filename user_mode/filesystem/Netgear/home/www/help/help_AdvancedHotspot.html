Hotspot Settings

Hotspot settings are useful when you are configuring a public Access Point. You can use Hotspot settings to redirect all HTTP requests to a Web server of your choice, through which you can manage the clients. For example, you can use the Web server for authentication, timing control, or advertising.

    • Enable HTTP Redirect: Enable this if you want all HTTP (TCP, port 80) requests to be "captured" and routed to the URL you specify in the next field.
    • URL: Enter the URL of the web server that you want all HTTP requests to be redirected to.
