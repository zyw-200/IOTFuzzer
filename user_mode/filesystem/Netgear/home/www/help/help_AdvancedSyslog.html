Syslog Settings

This page lets you configure the settings to connect to a Syslog server, if you have one configured in your network.

    • Enable Syslog: Enable the Syslog settings, if you have a Syslog server on your network.
    • Syslog Server IP Address: Enter the IP address to which the Access Point will send all SysLogs, if SysLog option is enabled.
    • Port: Enter the port number at which your Syslog server is configured to listen to requests.
