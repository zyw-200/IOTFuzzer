Mail Account Settings Help

  • From - Your mail box address.
  • To - Recipient's address.
  • SMTP Server - Your SMTP server.
  • Authentication - Most SMTP Server requires Authentication.
  • User Name - Your mail account name.
  • Password - Your mail account password.

Auto Mail Feature will help you monitor how your device is running.

  • Everyday, at specified time, this device will automatically send the log to specified mailbox.
  • Every few hours, such as 2 hours, this device will automatically send the log to specified mailbox.
